# StockBot - Gerenciamento de Estoque

Sistema full-stack de controle de estoque desenvolvido durante o curso Técnico em Desenvolvimento de Sistemas no SENAI.

## 🎯 Sobre o Projeto
Aplicação completa para cadastro, movimentação e consulta de produtos em estoque, com backend robusto e interface moderna.

## 🛠️ Tecnologias Utilizadas
- **Frontend**: React.js + Vite + JavaScript
- **Backend**: Java + Spring Boot (Spring Data JPA, Spring Security)
- **Banco de Dados**: MySQL
- **Outros**: Axios, React Router, Maven

## 🚀 Funcionalidades
- Cadastro, edição e exclusão de produtos
- Controle de entrada e saída de estoque
- Filtros e consultas dinâmicas
- Interface responsiva
